This repository contains the code for the [SpaceTour.com](https://spacetour.com) domain-for-sale landing page.

## Purpose

This landing page is designed to announce that SpaceTour.com is available for purchase and to collect inquiries from potential buyers.

## Features

- Modern, mobile-friendly single-page design
- Clearly states that SpaceTour.com is for sale
- Integrated inquiry form (works with Formspree or your chosen provider) for collecting offers and questions
- Automatically detects and displays the current domain name on the page and in form submissions
- Fast, secure, and easy deployment (static HTML, CSS, and JavaScript)
- Simple to customize for your needs

## Usage

1. **Clone or download this repository.**
2. **Edit the content** in `index.html` as needed (domain name, form endpoint, features, etc.).
3. **Deploy to your preferred static hosting provider** (Netlify, Vercel, GitHub Pages, cPanel, etc.).
4. **Point your DNS** so SpaceTour.com resolves to your hosting provider.

## Customization

- Update the domain name in the headers, content, and meta tags as needed
- Set the Formspree endpoint or other contact method for form submissions
- Adjust design, features, or text to match your preferences or branding

## DNS Setup

For best results, point SpaceTour.com to your hosting provider using a CNAME, ALIAS/ANAME, or A record as appropriate.  
Avoid HTTP redirects, as these can interfere with domain detection and form tracking.
